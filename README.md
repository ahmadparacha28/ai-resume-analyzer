# 🤖 AI Resume Analyzer

An AI-powered web application that analyzes resumes and matches them with job descriptions.

This project uses:
- Python (Flask) for backend
- NLP techniques for text processing
- React for frontend
- Machine Learning for similarity matching

---

## 🚀 Features

- Upload Resume (PDF or DOCX)
- Extract Skills Automatically
- Match Resume with Job Description
- Calculate Match Score
- Show Missing Skills (Skill Gap Analysis)
- Modern and Professional UI

---

### Backend:
